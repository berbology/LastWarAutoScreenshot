# Examples

This folder contains reference files for configuring Last War AutoScript.

## `ModuleConfig-example.jsonc`

An annotated copy of every configuration key at its default value, grouped by
`MouseControl`, `EmergencyStop`, `Screenshots` (including `SimilarityCheck`),
and `Logging` categories.

> **Note:** `.jsonc` files use JSON with Comments syntax and **cannot** be used
> directly as `ModuleConfig.json`. The real config file is created automatically
> with all defaults on first run — you do not need to copy this file anywhere.

Use this file as a lookup reference when you want to understand what a setting
does or what values it accepts. For the full reference see
[Docs/Configuration.md](../powershell-module/Docs/Configuration.md).
